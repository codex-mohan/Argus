# Kiro — AI-Assisted Development

## What It Provides

AI-powered development platform built on AWS. Speeds up prototyping with AI-assisted coding, full-stack app generation, and deployment tooling.

## How Argus Uses It

Kiro is a development accelerator, not a runtime dependency. Use it for:
- Rapid prototyping of the Next.js dashboard
- Generating boilerplate API routes
- AI-assisted refactoring during the build sprint

The final submission runs on Bun + Next.js + Docker, not on Kiro. Use Kiro during development, not as a deployment target.
